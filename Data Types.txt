Data Types

Numeric
- TinyInt				- 1 Byte
- SmallInt			- 2 Byte
- MediumInt			- 3 Byte
- Int						- 4 Byte
- BigInt				- 8 Byte
- Decimal				- fixed point decimal
- Float					- single precision floating point number
- Double				- double precision
- Bit						- single bit

Text

- Char					- fixed length string	(0-255)
- Varchar				- dynamic length string / variable length string (0-255)
- TinyText			
- Text
- MediumText
- LongText
Binary	- if we want to store images, pdf, or any other multimedia
- TinyBlob
- BLOB					- Binary Large Object
- MediumBlob
- LongBlob

- Time					- hh:mm:ss

		- requires 5 bytes of memory
- Timestamp			- CCYY-MM-DD hh:mm:ss	- 1970-01-01 00:00:01 to 2038:01-19 23:59:59
		- requires 4 btes of memory
